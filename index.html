CS61B BYOW

Build My Own World is an exciting project that utilizes Java and a variety of Java libraries to create a simulated world complete with inhabitants, landscapes, and ecosystems. The project offers an opportunity to implement and experiment with numerous algorithms, such as genetic algorithms, pathfinding algorithms, and neural networks. The result is a dynamic and ever-changing virtual world that can be customized and observed in real-time. With this project, you can explore the vast possibilities of Java programming and build your own world from scratch. Whether you're a seasoned developer or just starting, this project offers a fun and engaging challenge to hone your skills and creativity.

Universal Converter

The project I am working on is a conversion tool that can convert a variety of different things including units, currency, and numbers. This tool is designed to be easy to use and provide accurate results for all types of conversions. The user simply selects the type of conversion they would like to perform and enters the necessary information, such as the starting value and the desired unit or currency. The tool then performs the conversion and provides the user with the result. The project uses a variety of algorithms and libraries to ensure that all conversions are accurate and up-to-date with the latest exchange rates and unit conversions. Whether you are a student, a professional, or just someone who needs to convert values frequently, this tool is the perfect solution for all of your conversion needs.
